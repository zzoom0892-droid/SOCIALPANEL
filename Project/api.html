Public API

Endpoint: POST /api/order
Description: Place a new order.
Parameters:
  • category (string)
  • service (string)
  • link (string)
  • price (number)
Example Request: 
{
  "category": "Instagram",
  "service": "Followers",
  "link": "https://instagram.com/example",
  "price": 5
}
